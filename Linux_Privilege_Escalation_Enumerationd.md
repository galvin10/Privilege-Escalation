## Room Objective

This room focuses on Linux enumeration techniques used during privilege escalation assessments. The goal is to identify system information, users, services, scheduled tasks, network configurations, installed software, and sensitive files that may lead to privilege escalation opportunities.

### Skills Learned

- Operating System Enumeration
- Kernel Enumeration
- User & Group Enumeration
- Process Enumeration
- Network Enumeration
- Scheduled Task Enumeration
- Installed Package Enumeration
- Sensitive File Discovery
- SUID Binary Enumeration
- Linux Privilege Escalation Methodology

### Enumeration Workflow

1. Gather Operating System Information
2. Identify Users and Groups
3. Enumerate Running Processes
4. Check Scheduled Tasks (Cron Jobs)
5. Review Installed Packages
6. Enumerate Network Services
7. Discover Sensitive Files
8. Search for Writable Directories
9. Enumerate SUID Binaries
10. Identify Potential Privilege Escalation Vectors
